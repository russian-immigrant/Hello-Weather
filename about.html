About

Hello Weather is a simple static site for current conditions and a seven-day outlook anywhere you search.

Forecast and place search use the Open-Meteo API. Everything runs in your browser; no API key is required.

How to use

Open Current for live temperature, humidity, and wind, or 7-Day for the extended outlook. Your last city is saved locally in this browser.

Source code

The GitHub link is set in js/shar.js so you can change the repository URL in one place.
